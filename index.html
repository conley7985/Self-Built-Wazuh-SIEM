// 00 What This Lab Runs
SIEM + XDR, in one platform

Wazuh combines two disciplines that are usually sold as separate products. Here's what each one actually does.

Siem
Security Information & Event Management
Pulls logs from every connected system into one place, correlates events across sources, and scores each one by severity — so a single failed login is noise, but six from one IP in two minutes is a rule 10 alert.
Xdr
Extended Detection & Response
Goes past log analysis into the endpoint itself — file integrity monitoring, malware and rootkit checks, CVE scanning against installed software, and automated response actions like blocking an IP or killing a process.
Together
Detect, understand, and act
The SIEM layer tells you something happened and how severe it is. The XDR layer tells you where it happened on the actual machine, and gives the tools to respond — this lab exercises both ends of that loop.
